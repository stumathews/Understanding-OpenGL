Readme.txt
==========

cgtlib - Computer Games Technology Library provided by City University for Graphics Module INMxxx (static library)
Gamelib - Library that can be used in other C++ projects (Dynamic Link Library)
CourseWorkDemo - Game that was submitted to INMxxx Graphics Module
Stulib - Collection of classes used to package up common operations (used to create CourseWorkDemo) (static library)
Stulib2 - WIP - new version of Stulib such that Stulib still functions and so does CourseWorkDemo (which depends on Stulib and cgtlib) 
OpenGLn - Collection of Projects that are used as learning examples for training
